# GPT Image Playground 启动说明

本项目是一个基于 React + Vite 的前端应用。请按照以下步骤在本地启动：

## 1. 环境准备
确保你的电脑上已经安装了 **Node.js** (推荐使用 v18 或更高版本)。

## 2. 安装依赖
在项目根目录下打开终端，运行以下命令安装必要的依赖包：

```bash
npm install
```

## 3. 启动项目
安装完成后，运行以下命令启动开发服务器：

```bash
npm run dev
```

## 4. 访问应用
启动成功后，终端会显示访问地址，通常是：
[http://localhost:5173](http://localhost:5173)

打开浏览器访问该地址即可开始使用。

## 5. 配置 API
1. 进入页面后，点击右上角的 **设置 (齿轮图标)**。
2. 在弹出的窗口中配置你的 **API URL** (例如 `https://api.openai.com`)。
3. 填入你的 **API Key**。
4. 保存后即可开始生成或编辑图片。

---

### 其他常用命令
- `npm run build`: 构建生产环境产物（输出到 `dist` 目录）。
- `npm run preview`: 在本地预览构建后的产物。
